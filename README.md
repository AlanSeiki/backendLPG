## Descrição

Backend para um aplicativo de finanças pessoais Little Pig Golden.

## Tecnologias
```bash
 $ Nest.js
 $ TypeScript
 $ TypeOrm
 $ PostgreSQL
```
## Instalação

```bash
$ npm install
$ npm run typeorm:run
$ npm run typeorm-seed:run
```

## Inicializar o serviço

```bash
# development
$ npm run start
```
