## Descrição

Aplicativo de finanças pessoais.

## Tecnologias

Nest.js
TypeScript
TypeOrm
PostgresSQL

## Instalação

```bash
$ npm install
$ npm run typeorm:run
$ npm run typeorm-seed:run
```

## Inicializar o serviço

```bash
# development
$ npm run start
```
